# CLOUD-DATA-SECURITY
The Problem: Traditional cloud storage uses centralized key management, making it highly vulnerable to data breaches and unauthorized access.
The Solution: A three-phase hybrid security framework designed to protect cloud data at every level.
Phase 1 (Dynamic AES): Encrypts every single uploaded file with a unique, dynamically generated key.
Phase 2 (Blockchain): Stores those encryption keys in a decentralized, tamper-proof blockchain network to eliminate single points of failure.
Phase 3 (ECC): Uses Elliptic Curve Cryptography to securely and efficiently transmit the data and keys between users.
The Outcome: A highly secure, scalable, and efficient standard for protecting sensitive information in modern cloud systems.
